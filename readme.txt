This plugin will help you to create a scroll to top button on your site

== Description ==

This is a simple and yet awesome plugin to enabling scroll top functionality in your site. After install the plugin you will immediately see a rocket image button on the right bottom side of you website. And you can simply customize some settings for that plugin.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plugin files to your plugin directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->SS Awesome Scroll Top to configure the plugin or you will see a settings link while activating the plugin. That link will take to that plugin options page. 


== Frequently Asked Questions ==

= Can I change rocket image? =

Yes you can change rocket images. There has three differect rocket images where from you can choose.

= Can I change button image opacity, scroll offset, scroll speed? =

Yes ofcouse, you can change then from plugin options.
